# Calendrier d'Alimentation — Beaufort PWA

## Installation sur Android

1. Déposez ce dossier sur **netlify.com** (glisser-déposer, gratuit)
2. Ouvrez l'URL générée dans **Chrome Android**
3. Menu ⋮ → **"Ajouter à l'écran d'accueil"**
4. L'app s'installe comme une vraie application

## Fichiers
- `index.html` — Application complète
- `manifest.json` — Identité PWA (nom, icône, couleurs)
- `sw.js` — Service worker (fonctionne hors-ligne)
- `icon.svg` — Icône de l'application

## Fonctionnalités
- Calendrier d'alimentation par périodes quinzaine
- Gestion multi-campagnes (2023–2030)
- Onglet dates troupeau
- Suivi production laitière avec calculs L/vache/an
- Graphiques d'évolution
- Export PDF A4, CSV, JSON
- Sauvegarde automatique locale
- Fonctionne hors-ligne
